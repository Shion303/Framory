# Roadmap Esecutiva

## Milestone 1 - Fondazioni

Next.js, React, TypeScript strict, Tailwind, Prisma, env validation, layout, UI base, README, CI, test iniziali.

## Milestone 2 - Autenticazione

Registrazione, login, logout, sessioni, cookie sicuri, rate limiting, ruoli, owner seed e route protette.

## Milestone 3 - Catalogo Admin

CRUD controllato per franchise, collection, work, season, episode, conferme e audit log.

## Milestone 4 - AniList

Ricerca GraphQL e import controllato di una singola opera in un franchise scelto.

## Milestone 5-9 - Esperienza utente

Discovery, libreria, tracking episodi, pagina franchise e home con prossimo episodio.

## Milestone 10-12 - Comunità e controllo

Profilo/privacy, badge, attività, report, moderazione, gestione utenti, impostazioni piattaforma.

## Milestone 13-15 - Qualità e deploy

Responsive, accessibilità, stati UI, sicurezza, test end-to-end, guide deploy e manutenzione.
